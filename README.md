<h1>Vůně dálek – e-shop s kořením z Dubaje</h1>

<h2>Popis projektu</h2>
<p>
    Tento projekt představuje webovou aplikaci (e-shop) s názvem 
    <strong>Vůně dálek</strong>, která se zaměřuje na prodej exotického koření dováženého z Dubaje do České republiky.
</p>
<p>
    Cílem webu je nabídnout uživatelům přehledný katalog kvalitního koření, umožnit detailní zobrazení jednotlivých produktů 
    a vytvořit příjemný zážitek z nakupování inspirovaný orientální atmosférou.
</p>

<h2>Cíl webu</h2>
<ul>
    <li>prezentovat firmu a její příběh</li>
    <li>nabídnout přehledný katalog koření</li>
    <li>umožnit uživateli zobrazit detail produktu</li>
    <li>vytvořit základ pro jednoduchý e-shop</li>
</ul>

<h2>Struktura webu</h2>
<p>Web se skládá z následujících hlavních částí:</p>

<h3>Hlavní stránka (Home)</h3>
<ul>
    <li>představení firmy Vůně dálek</li>
    <li>informace o dovozu koření z Dubaje</li>
    <li>úvodní vizuál a atmosféra značky</li>
</ul>

<h3>Katalog produktů</h3>
<ul>
    <li>seznam dostupného koření (např. kurkuma, skořice, kari, sušená cibule apod.)</li>
    <li>přehledné zobrazení produktů</li>
    <li>možnost kliknutí na detail produktu</li>
</ul>

<h3>Detail produktu</h3>
<ul>
    <li>název koření</li>
    <li>popis a původ</li>
    <li>obrázek produktu</li>
    <li>případně cena a další informace</li>
</ul>

<h2>Očekávaný obsah</h2>
<ul>
    <li>exotické koření z Dubaje</li>
    <li>produktové fotografie</li>
    <li>popisy koření a jejich využití</li>
    <li>základní informace o firmě</li>
</ul>
